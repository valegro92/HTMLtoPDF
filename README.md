# HTML → PDF Converter

Trasforma qualsiasi codice HTML in un PDF impaginato correttamente, con anteprima live in tempo reale.

Uno strumento gratuito de **[La Cassetta degli AI-trezzi](https://lacassettadegliaitrezzi.substack.com/)** — la newsletter sull'adozione AI nelle PMI italiane.

## Come funziona

1. Incolla il tuo HTML nel pannello sinistro
2. L'anteprima si aggiorna istantaneamente nel pannello destro
3. Scegli il formato e clicca **Genera PDF**

## Formati disponibili

| Formato | Ideale per |
|---|---|
| **Auto** | Infografiche, poster, banner — il PDF si adatta al contenuto |
| **A4** | Documenti, report, lettere |
| **Letter** | Formato US standard |
| **A3** | Poster e stampe grandi |

Con l'opzione **Adatta a pagina**, il contenuto viene scalato automaticamente per entrare nel formato scelto.

## Stack

- **Express.js** + **Puppeteer** (Chromium headless)
- Frontend vanilla HTML/CSS/JS — un singolo file, zero dipendenze frontend
- Deploy-ready su Render.com (free tier)
